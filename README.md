# Time Pass Korea

> **AI 기반 국가유산 맞춤형 해설 가이드 플랫폼**  
> 외래 관광객이 한국의 국가유산을 자신의 언어, 문화권, 관심사, 관람 목적에 맞게 이해하고 체험할 수 있도록 돕는 AI 도슨트 서비스입니다.

<br />

## 1. 프로젝트 개요

**Time Pass Korea**는 한국 역사와 문화가 낯선 외래 관광객을 위한 **AI 기반 국가유산 해설 서비스**입니다.

기존 문화유산 해설 서비스는 정해진 시간, 정해진 언어, 정해진 방식으로 제공되는 경우가 많아 사용자가 원하는 순간에 자유롭게 이용하기 어렵습니다. 또한 단순 텍스트나 녹음 해설 중심이라 실제 현장에서 보고 있는 유물과 상호작용하기 어렵다는 한계가 있습니다.

이 프로젝트는 이러한 문제를 해결하기 위해 다음과 같은 경험을 제공합니다.

- QR 또는 GPS 기반으로 별도 앱 설치 없이 즉시 진입
- 국가, 연령, 관심사에 따른 맞춤형 해설 모드 추천
- 카메라 기반 문화유산 이미지 탐지
- AI 챗봇을 통한 실시간 질의응답
- 현재 위치 기반 문화유산, 포토스팟, 관람 동선 추천
- 여행자, 학자, 어린이/가족 관람객을 위한 차별화된 해설 경험

<br />

## 2. 문제 정의

방한 외래 관광객은 증가하고 있으며, 고궁과 역사유적지에 대한 관심도도 높습니다. 그러나 실제 관람 경험은 여전히 다음과 같은 문제를 가지고 있습니다.

### 기존 서비스의 한계

| 문제                 | 설명                                                                   |
| -------------------- | ---------------------------------------------------------------------- |
| 제한된 이용 시간     | 공식 해설은 정해진 시간과 회차 중심으로 운영되어 자유 관람과 맞지 않음 |
| 언어 장벽            | 한국어 또는 전문 용어 중심 설명으로 외국인 관광객의 이해도 저하        |
| 일방향 해설          | 사전 녹음, 정적 텍스트 중심으로 사용자의 질문이나 관심사 반영이 어려움 |
| 낮은 몰입도          | 스탬프, 경품 중심의 관람은 실제 문화유산 이해로 이어지기 어려움        |
| 현장 상호작용 부족   | GPS 위치 안내는 가능하지만 실제 보고 있는 유물과의 연결성은 약함       |
| 사용자별 차별화 부족 | 어린이, 일반 관광객, 역사 마니아에게 동일한 난이도의 콘텐츠 제공       |

<br />

## 3. 해결 방향

Time Pass Korea는 **“같은 유산, 다른 경험”**이라는 콘셉트 아래, 사용자의 배경과 목적에 따라 문화유산 해설을 다르게 제공합니다.

### 핵심 해결 전략

1. **사용자 맞춤형 데이터 큐레이션**  
   사용자가 입력한 국가, 연령, 관심사를 기반으로 적합한 관람 모드를 추천합니다.

2. **AI 문화 번역**  
   단순 직역이 아니라, 외국인 관광객이 이해하기 쉬운 문화적 맥락으로 한국 역사와 국가유산을 설명합니다.

3. **현장 기반 참여형 관람**  
   사용자가 직접 촬영하고, 질문하고, 이동하면서 문화유산을 탐색할 수 있는 체험형 구조를 제공합니다.

4. **모드별 해설 경험 제공**  
   여행자, 학자, 어린이/가족 등 사용자 유형에 따라 다른 깊이와 방식의 해설을 제공합니다.

<br />

## 4. 주요 기능

## 4.1 QR / GPS 기반 진입

사용자는 현장에 배치된 QR 코드를 스캔하거나 현재 위치 기반으로 서비스에 접속할 수 있습니다.  
별도의 앱 설치 없이 모바일 브라우저에서 바로 이용할 수 있도록 설계했습니다.

<br />

## 4.2 사용자 맞춤형 3-way 해설 모드

### 여행자 모드

일반 관광객과 K-컬처 팬덤 사용자를 위한 기본 모드입니다.

- 핵심 랜드마크 중심 해설
- 효율적인 관람 동선 추천
- 포토스팟 추천
- 짧고 직관적인 설명 제공
- 퀘스트 기반 현장 체험 제공

### 학자 모드

깊이 있는 역사와 구조적 설명을 원하는 사용자를 위한 모드입니다.

- 국가유산청 공공데이터 기반 전문 정보 제공
- 건축 구조, 역사적 배경, 문화적 의미 중심 해설
- 심화 아카이브형 콘텐츠 제공
- 환각을 줄이기 위한 검증 데이터 기반 응답 구조 지향

### 동화 모드

어린이 및 가족 단위 관람객을 위한 모드입니다.

- 어린이 눈높이에 맞춘 스토리텔링 해설
- 퀴즈형 참여 콘텐츠
- 탐험 지도 기반 미션
- 문화유산을 놀이처럼 학습할 수 있는 경험 제공

<br />

## 4.3 이미지 탐지 기반 해설

사용자가 문화유산이나 유물을 촬영하면 AI가 이미지를 분석해 어떤 대상인지 인식합니다.

인식된 결과를 바탕으로 다음 정보를 제공합니다.

- 유물 또는 건축물 이름
- 역사적 배경
- 건축적 특징
- 숨은 이야기
- 사용자의 모드에 맞춘 설명

<br />

## 4.4 AI 챗봇

사용자는 관람 중 궁금한 내용을 실시간으로 질문할 수 있습니다.

예시 질문:

```text
이 건물은 왜 왕이 사용했나요?
프랑스 궁전과 비교하면 어떤 의미가 있나요?
아이에게 쉽게 설명하려면 어떻게 말할 수 있나요?
이 장소에서 사진을 찍기 좋은 위치는 어디인가요?
```

````

AI 챗봇은 사용자의 언어, 관심사, 관람 모드를 반영해 맞춤형 답변을 제공합니다.

<br />

## 4.5 GPS 기반 추천

현재 위치를 기반으로 가까운 국가유산, 추천 동선, 포토스팟을 안내합니다.

주요 기능:

- 현재 위치 기반 주변 문화유산 추천
- 가까운 관람 스팟 안내
- 포토스팟 추천
- 이동 경로 기반 다음 관람 지점 제안
- 경복궁, 경주 등 공간 특성에 따른 맞춤형 경험 제공

<br />

## 5. 사용자 여정

```text
1. QR 또는 GPS로 서비스 진입
2. 국가, 연령, 관심사 입력

5. 카메라로 유물 또는 건축물 촬영

7. 사용자가 챗봇에 추가 질문
8. 퀘스트, 포토스팟, 도장 기록 등 체험 완료
9. 관람 리포트 저장
```

## 10. 기대 효과

### 10.1 외래 관광객의 문화유산 이해도 향상

사용자의 문화권과 관심사에 맞는 설명을 제공하여, 한국 역사와 국가유산을 더 쉽게 이해할 수 있도록 돕습니다.

### 10.2 관람 경험의 질적 개선

단순히 보는 관람에서 벗어나 촬영, 질문, 이동, 퀘스트 수행이 결합된 능동적 관람 경험을 제공합니다.

### 10.3 국가유산 관광 콘텐츠 확장

경복궁과 같은 궁궐형 공간뿐만 아니라 경주처럼 도시 전체가 문화유산인 지역까지 확장할 수 있습니다.

### 10.4 K-콘텐츠와 로컬 관광 연계

드라마 촬영지, 야간 행사, 지역 축제 등과 연결하여 외래 관광객의 관광 경험을 문화유산 밖으로 확장할 수 있습니다.

<br />

## 11. 향후 확장 방향

### 11.1 국가별 문화 맥락 비교 해설

예를 들어 프랑스 사용자에게는 근정전을 베르사유 궁전과 비교해 설명하는 방식으로, 사용자의 국가적 배경에 맞는 문화 비교 해설을 제공할 수 있습니다.

```text
기존 해설:
근정전은 조선 왕조의 정전으로, 국가 의례와 공식 행사가 거행되던 궁궐의 중심 건축물입니다.

확장 해설:
베르사유 궁전이 프랑스 왕권의 상징이듯, 근정전은 조선 왕권과 국가 의례의 중심 공간입니다.
```

### 11.2 전국 단위 국가유산 플랫폼 확장

- 서울 궁궐권
- 경주 역사유적지구
- 전주 한옥마을
- 공주·부여 백제문화권
- 지역 축제 및 야간 관광 콘텐츠

### 11.3 사용자 피드백 기반 추천 고도화

사용자의 관람 기록과 피드백이 쌓일수록 추천 동선과 해설 품질을 개선하는 자가 발전형 구조로 확장할 수 있습니다.

- 외국인 관광객이 한국의 역사적 맥락을 어떻게 쉽게 이해할 수 있을까?
- 국가유산 해설을 정적인 정보 전달이 아니라 참여형 경험으로 바꿀 수 있을까?
- AI 이미지 탐지와 LLM 기반 해설을 실제 사용자 여정 안에서 어떻게 자연스럽게 연결할 수 있을까?
- 관광, 교육, 문화유산 보존을 함께 고려한 서비스로 확장할 수 있을까?

Time Pass Korea는 기술적으로는 Vue, Django, YOLOv8, LLM API를 결합한 AI 융합 웹 서비스이며, 서비스적으로는 한국 국가유산의 접근성과 몰입도를 높이는 것을 목표로 합니다.
````
